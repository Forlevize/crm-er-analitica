# ER Analitica

Sistema web para gestao de equipamentos e calibracoes da ER Analitica para a Veolia.

## Stack
- React + TypeScript + Vite
- TailwindCSS
- Supabase (Auth + Postgres + Edge Functions)
- Recharts, dnd-kit, jsPDF, xlsx

## Estrutura
- `src/`: frontend
- `supabase/`: migrations, seed e Edge Functions
- `docs/`: arquitetura, permissoes e deploy

## Scripts
- `npm run dev`
- `npm run build`
- `npm run test`

## Observacoes
- Este scaffold inclui contratos, telas e integracoes prontas para conectar a um projeto Supabase.
- As dependencias nao foram instaladas automaticamente neste ambiente; execute `npm install` antes de rodar localmente.
